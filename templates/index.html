About CreditWise AI

CreditWise AI is an intelligent Credit Card Approval Prediction platform developed using Machine Learning and Flask.

The system analyzes applicant information such as income, employment, education, housing and family details to predict whether a credit card application is likely to be approved.

This project demonstrates an end-to-end Machine Learning workflow including data preprocessing, model training, evaluation and real-time deployment.
